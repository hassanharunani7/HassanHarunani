## 🔧 Customization Guide

### Change Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #00a8e8;
    --secondary-color: #004e89;
    --accent-color: #00c9a7;
    /* ... etc */
}
```

### Add Your Photos
Replace placeholder sections with actual images:
```html
<img src="your-image.jpg" alt="Description">
```

### Update Content
Edit text content directly in `index.html` sections.

### Add Social Links
Update footer social links in `index.html`:
```html
<a href="https://your-social-link" class="social-link">
    <i class="fab fa-platform"></i>
</a>
```
